# Green Transport Patent Intelligence Pipeline

A full end-to-end data pipeline analysing **141,124 active green transportation patents** (CPC Y02T) granted by the USPTO from 1976 to 2025.

**Focus:** Climate change mitigation in transportation — covering electric vehicles, hybrid engines, fuel cells, aviation propulsion, maritime shipping, and charging infrastructure.

---

## Pipeline Architecture

```
USPTO PatentsView (manual download → data/raw/)
↓
00_check_raw.py      — validates raw file columns
↓
02_clean.py          — filters Y02T patents, cleans data
↓
02b_data_quality.py  — quality checks, fixes geocoding errors
↓
03_load_db.py        — loads 4 tables into SQLite (patents.db)
↓
04_analyze.py        — runs 7 SQL queries, exports CSVs
↓
05_report.py         — console report + JSON + summary CSV
↓
dashboard/app.py     — Streamlit interactive dashboard
```

## Quickstart

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/green-patent-pipeline.git
cd green-patent-pipeline
```

### 2. Create virtual environment and install dependencies

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r green-patent-pipeline/requirements.txt
```

### 3. Download raw data files (REQUIRED)

> The raw data files total ~13 GB and cannot be stored on GitHub.
> They must be downloaded manually before running the pipeline.

**Instructions:**

1. Open your browser and go to:
   <https://data.uspto.gov/bulkdata/datasets/pvgpatdis>

2. On the page, set the date range:
   - **From:** 1976-01-01
   - **To:** 2025-09-30

3. Download and **unzip** each of these files:

   | File | Size | Description |
   |------|------|-------------|
   | `g_patent.tsv` | ~1.1 GB | Core patent info |
   | `g_patent_abstract.tsv` | ~5.9 GB | Patent abstracts |
   | `g_inventor_disambiguated.tsv` | ~2.2 GB | Inventor names & locations |
   | `g_assignee_disambiguated.tsv` | ~1.1 GB | Company/assignee names |
   | `g_location_disambiguated.tsv` | ~9 MB | City, state, country lookup |
   | `g_cpc_current.tsv` | ~3.2 GB | CPC classification codes |

4. Place all 6 unzipped `.tsv` files into: `green-patent-pipeline/data/raw/`

   Your folder should look like this:

   ```
   green-patent-pipeline/data/raw/
   ├── g_patent.tsv
   ├── g_patent_abstract.tsv
   ├── g_inventor_disambiguated.tsv
   ├── g_assignee_disambiguated.tsv
   ├── g_location_disambiguated.tsv
   └── g_cpc_current.tsv
   ```

> **Note:** These files are listed in `.gitignore` and will never be committed to GitHub. You only need to download them once.

### 4. Run the full pipeline

```bash
cd green-patent-pipeline
python3 scripts/run_pipeline.py
```

This runs all 6 steps automatically (~6 minutes):

| Step | Script | What it does |
|------|--------|-------------|
| 0 | `00_check_raw.py` | Verifies all 6 raw files have correct columns |
| 1 | `02_clean.py` | Filters 141,124 Y02T patents, cleans & saves CSVs |
| 2 | `02b_data_quality.py` | Quality checks, fixes 5 geocoding errors |
| 3 | `03_load_db.py` | Loads 4 tables into SQLite, creates 9 indexes |
| 4 | `04_analyze.py` | Runs 7 SQL queries, exports results to CSV |
| 5 | `05_report.py` | Generates console report, JSON, summary CSV |

A complete log is saved to `green-patent-pipeline/output/logs/pipeline_latest.txt`.

### 5. Launch the interactive dashboard

```bash
cd green-patent-pipeline
streamlit run dashboard/app.py
```

Open your browser at **<http://localhost:8501>**

## Database Schema

| Table | Rows | Purpose |
|-------|------|---------|
| `patents` | 141,124 | Core patent metadata (ID, title, abstract, dates) |
| `inventors` | 138,160 | Inventor profiles (name, country, location) |
| `companies` | 12,201 | Company/assignee profiles (name, type, location) |
| `relations` | 419,757 | Links patents to inventors and companies |

---

## The 7 SQL Queries

| Query | Description | Technique |
|-------|-------------|-----------|
| Q1 | Top inventors by patent count | GROUP BY, COUNT DISTINCT |
| Q2 | Top companies by patent count | GROUP BY, COUNT DISTINCT |
| Q3a | Countries by inventor location | Window function, percentage |
| Q3b | Countries by company location | Window function, percentage |
| Q4 | Patent trends over time | Cumulative SUM window |
| Q5 | Full JOIN across all 4 tables | Multi-table JOIN |
| Q6 | Above-average companies | 3-step CTE (WITH clause) |
| Q7 | Inventor rankings | RANK, DENSE_RANK, window partition |

## Data Quality

The pipeline includes a dedicated quality check step (`02b_data_quality.py`) that performs 17 checks. Key findings:

- ✅ Zero duplicate primary keys
- ✅ Zero orphaned foreign keys
- ✅ 5 geocoding errors identified and corrected (Japanese entities misclassified as Uganda/Cameroon)
- ⚠️ 0.7% missing inventor country (kept as-is)
- ⚠️ 1.7% missing company country (kept as-is)

## Dashboard Features

The Streamlit dashboard provides 6 interactive pages:

1. **Overview** — Key metrics, patent growth, top countries, subcategory breakdown
2. **Trends Over Time** — Annual filings, cumulative growth, year-range filtering
3. **Top Inventors** — Prolific inventors, country distribution, rankings
4. **Top Companies** — Leading patent holders, country breakdown
5. **Country Analysis** — Inventor vs company locations, technology transfer patterns
6. **Patent Explorer** — Search patents by title/ID and year range
